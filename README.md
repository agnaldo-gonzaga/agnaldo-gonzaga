### 🚀 Sobre mim

🎓 Graduando em **Ciência de Dados**, com foco em transformar dados operacionais em inteligência estratégica.

🌱 Atualmente atuo como **Analista de Controle Agrícola**, onde aplico análise de dados para otimização de processos críticos (CTT - Corte, Transbordo e Transporte). Minha experiência inclui:

- Desenvolvimento de Dashboards estratégicos (Power BI/Excel)
- Monitoramento de KPIs de frota e logística em tempo real
- Estruturação de dados para tomada de decisão baseada em evidências
- Automação de rotinas analíticas com SQL e Python

Hoje, foco minha evolução na **Modern Data Stack** (dbt, Cloud Data Warehousing) e em modelos de **Machine Learning** aplicados ao negócio.
