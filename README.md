# 🚀 Sobre mim

🎓 Graduando em **Ciência de Dados**, com foco em transformar dados operacionais em inteligência estratégica.

🌱 Atualmente atuo como **Analista de Controle Agrícola**, utilizando análise de dados para otimização de processos ligados ao **CTT (Corte, Transbordo e Transporte)**.

Minha experiência inclui:

- 📊 Desenvolvimento de dashboards estratégicos (Power BI e Excel)
- 🚛 Monitoramento de KPIs logísticos e de frota em tempo real
- 🧠 Estruturação de dados para tomada de decisão baseada em evidências
- ⚡ Automação de rotinas analíticas com SQL e Python

Atualmente, estou aprofundando meus conhecimentos em:

- Analytics Engineering
- Engenharia de Dados
- Cloud & Data Warehousing
- Machine Learning aplicado ao negócio

Meu objetivo é unir tecnologia, analytics e agronegócio para gerar eficiência operacional e inteligência estratégica através dos dados.
